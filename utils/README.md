# AI-Based Applicant Tracking System (ATS)

A full-stack web application designed to streamline the hiring process by helping HR teams create job postings, upload resumes, analyze candidates using AI, and manage recruitment pipelines efficiently.

## Features

- User Authentication (HR / Manager roles)
- Create and Manage Job Postings
- Resume Upload (PDF / DOCX)
- AI Resume Parsing and Candidate Scoring
- Skill Matching Against Job Requirements
- Candidate Pipeline Management
- Dashboard with Hiring Insights
- Responsive Modern UI

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

### AI Module
- Python
- Resume Parsing
- Keyword Matching
- Basic NLP
